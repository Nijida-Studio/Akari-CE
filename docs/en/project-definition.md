# Project definition

## Current phase

Akari CE is being prepared as an independent software project. The next step is not implementation of individual features, but a sound definition of the project.

## Next tasks

1. Describe the problem and intended audience.
2. Define the scope and explicit non-goals.
3. Establish components and responsibility boundaries.
4. Document interfaces with other Dokoni projects.
5. Agree on architecture, development conventions, and an initial roadmap.

## Documentation principle

Confirmed public product information belongs in this repository. Preliminary working knowledge should not accidentally constrain the eventual architecture.
