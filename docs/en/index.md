# Akari CE

Akari CE is an independent software project within the Dokoni ecosystem. It is currently in an early definition and planning phase.

## What is established

- Akari CE has an independent lifecycle and repository.
- Coupling to other Dokoni projects should be explicit and kept as limited as practical.
- AMTS may organize project knowledge but does not define the product or its software architecture.
- The project is intended to be developed and documented as open-source software.

## What remains open

The detailed product purpose, feature scope, components, interfaces, architecture, and roadmap have not yet been agreed. This documentation makes the early status visible without pre-empting future decisions.
