# Akari CE

Akari CEは、Dokoniエコシステムに属する独立したソフトウェアプロジェクトです。現在は、目的や構成を検討する初期段階にあります。

## 現時点で決まっていること

- Akari CEは独立したライフサイクルとリポジトリを持ちます。
- 他のDokoniプロジェクトとの関係は明示し、必要以上に強く結び付けません。
- AMTSをプロジェクト知識の整理に利用することはありますが、製品やソフトウェア構成をAMTSが定めることはありません。
- オープンソースソフトウェアとして開発し、文書を公開する方針です。

## これから決めること

具体的な目的、機能の範囲、構成要素、インターフェース、アーキテクチャ、ロードマップはまだ確定していません。今後の判断を先取りせず、現在の段階を正直にお伝えします。

続きは[プロジェクトの定義](project-definition.md)をご覧ください。
