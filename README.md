# Akari CE

Akari CE ist ein eigenständiges Open-Source-Softwareprojekt im Dokoni-Ökosystem. Das Projekt befindet sich in der Definitionsphase: Anwendungsbereich, Komponenten, Schnittstellen und Roadmap werden derzeit erarbeitet.

Diese frühe Dokumentation hält bewusst nur bestätigte Aussagen fest. Sie wird erweitert, sobald Produktzweck und Architektur belastbar entschieden sind.

## Beziehung zu AMTS

Akari CE kann AMTS zur Organisation von Projektwissen verwenden. AMTS ist jedoch eine unabhängige Spezifikation und keine Softwareabhängigkeit von Akari CE.

## Lizenz

Akari CE ist als freie Open-Source-Software unter der Apache License 2.0 vorgesehen. Maßgeblich ist die im Repository veröffentlichte Lizenzdatei.
